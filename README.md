# Sistema de Triagem de Pronto Socorro

Projeto backend em Java 17 com Spring Boot para gerenciar triagem de pacientes em pronto socorro.

## Tecnologias

- Java 17+
- Spring Boot
- Maven
- Banco de dados H2 (em memória)
- JPA / Hibernate
- API RESTful com JSON
